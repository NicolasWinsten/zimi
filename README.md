# ZiMi 字谜!
Zimi (character riddle) is a word matching game for Chinese learners (and Chinese learned-ers). Each day brings a new puzzle: a grid of Chinese characters that must be matched to form words.  Click on two tiles to match them if they form a word. Finish the puzzle by matching all the characters without making 3 mismatches. Complete the puzzle each day to earn your streak.
